Step 1 - Choose a SQL Datasource

APIGen works by analyzing SQL object data (tables, columns, keys, etc..) and mapping this data to template files which are copied to an output directory.

You can provide a SQL schema file, a create database script or a connection string to allow APIGen to analyze the data.

To view a working example, choose one of the example databases from the SQL Mgmt Web app and click the Copy SQL Connection String button in the upper right hand corner.
